irregulares nessas matrizes irregulares no java isso vai depender muito da linguagem de programação que você está trabalhando mas o java permite que
você tenha arrays irregulares ou matrizes irregulares, o que significa isso? na tela por exemplo nós sabemos que essa matriz tem três linhas nem tão
nos vamos as linhas onde posições eram 1 e 2 porém cada linha em diferentes colunas, então por exemplo a primeira linha ela tem somente uma coluna a
segunda linha tem somente duas colunas e a terceira linha tem três colunas então esse tipo de estrutura de dado você consegue falar e você tem no
java também, então o java permite isso justamente quando nós trabalhamos com arrays nós estamos trabalhando com objetos, então justamente por que
o java trata os arrays como objetos a gente consegue fazer isso como as matrizes elas são arrays de arrays elas são vetores de vetores e justamente

Como vamos fazer

Escreva um programa para guardar dados de uma
pesquisa feita com uma amostra de pessoas da
avenida mais movimentada da cidade onde você
mora.

A pesquisa consiste em perguntar ao cidadao a
quantidade de filhos e o nome de cada filho. Guarde
essas informagdes em uma matriz de forma que
cada pessoa entrevista ocupe somente 0 espaco
necessario na memoria para guardar os nomes dos
filhos.

Imprima a quantidade de pessoas entrevistadas, a
quantidade de filhos de cada entrevistado e os
respectivos nomes dos filhos.


Ex:

package com.madu.Matrizes.arrays;

import java.util.Scanner;

public class MatrizIrregular {

	public static void main(String[] args) {
		
		Scanner scan = new Scanner(System.in);
		
		System.out.println("Entre com o número de pessoas que serão entrevistadas:");
		int numEntrevistados = scan.nextInt();
		
		String[][] nomeFilhos = new String[numEntrevistados][];
		
		for (int i=0; i<nomeFilhos.length; i++) {
			
			System.out.println("Entre com a contidade de filhos");
			int qtdFilhos = scan.nextInt();
			
			nomeFilhos[i] = new String[qtdFilhos];
			
			for (int j=0; j<nomeFilhos[i].length; j++) {
				
				System.out.println("Digite o nome do filho " + (j+1));
				
				nomeFilhos[i][j] = scan.next();
			}
		}
		
		for (int i=0; i<nomeFilhos.length; i++) {
			System.out.println("Pessoa " + i + " tem " + nomeFilhos[i].length + " filhos");
			for (int j=0; j<nomeFilhos[i].length; j++) {
				System.out.println(nomeFilhos[i][j]);
			}
		}
	}
}
